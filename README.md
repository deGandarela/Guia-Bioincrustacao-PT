# Biofouling ID Guide - Global

This repository contains a quick-reference web application designed for commercial divers, hull cleaners, marine inspectors, and citizen scientists worldwide. Its primary purpose is to help users easily identify and report marine invasive species (biofouling) found on ship hulls and niche areas (like sea chests and grates) before they are scraped off or spread across international waters.

## Features

* **Quick Identification:** Simple visual cards detailing key identification features for over 45 globally recognized invasive species.
* **Risk Categorization:** Species are categorized into Primary Invasive (Hull Foulers) and Niche Area Hitchhikers (Hidden).
* **Global Geographic Filtering:** Dynamic dropdown menus to filter invasive species based on specific global marine regions (Europe Seas, Americas & Caribbean, Asia & Indo-Pacific, and Africa & Middle East), plus a quick "Global (All Waters)" reset button.
* **Interactive Gallery:** Clickable cards open a modal with detailed descriptions, multiple images, and identification warnings.
* **Reporting Guidelines:** Built-in instructions and contacts for Local Port Authorities, the IUCN Global Invasive Species Database, and the IMO GloFouling project to ensure rapid reporting.

## Data Source

The species listed, categorized, and geographically mapped in this guide are rigorously cross-referenced using international marine pest tracking databases, including:
* The IUCN Global Invasive Species Database (GISD)
* The Smithsonian's National Exotic Marine and Estuarine Species Information System (NEMESIS)
* The International Maritime Organization (IMO) GloFouling Guidelines
* The World Register of Marine Species (WoRMS)
